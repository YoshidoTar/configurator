Конфигуратор выполняет узконаправленную задачу по сборке персональных компьютеров и ориентирован на пользователей, не обладающих специализированными знаниями в области компьютерного оборудования. Приложение предназначено для максимально простого и понятного подбора конфигурации ПК без необходимости глубокого понимания принципов работы комплектующих.
Конфигуратор РАНХиГС — это программный продукт для подбора персональных компьютеров, который объединяет в себе:
удобный и интуитивно понятный инструмент сборки ПК;
встроенную систему помощи для пользователей любого уровня подготовки;
каталог готовых, проверенных и сбалансированных конфигураций.
Проект разработан с целью упрощения процесса выбора компьютерных комплектующих, обеспечения их совместимости и снижения вероятности ошибок при формировании конфигурации.
Цели проекта:
  создание интуитивно понятного инструмента для сборки персональных компьютеров;
  обучение пользователей базовым принципам совместимости компьютерных компонентов;
  сокращение времени, необходимого для подбора конфигурации;
  предоставление экспертных готовых решений для различных задач и бюджетов;
  минимизация ошибок при выборе и комбинировании комплектующих.
